## Server酱

* 1.手机或电脑浏览器登陆[Server酱官网](https://sct.ftqq.com/login)
* 2.VX扫码（注意关闭免打扰）
* 3.返回手机或电脑浏览器点击继续
* 4.复制SendKey填入`config.json`文件中
